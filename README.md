# Практична робота: Основи CSS. Оформлення багатосторінкового HTML-сайту

## Опис
Цей проєкт є логічним продовженням першої практичної роботи. Головна мета — закріпити навички підключення зовнішніх таблиць стилів та візуального оформлення багатосторінкового сайту засобами CSS.

## Що було реалізовано:
- Підключено зовнішні CSS-файли (style.css та responsive.css) до всіх трьох сторінок.
- Створено дизайн-систему: налаштовано базову кольорову палітру (через змінні :root), типографіку та глобальні стилі.
- Використано Flexbox для створення горизонтального меню навігації та розміщення медіа-елементів на сторінці "Про мене".
- Стилізовано текстовий контент, списки, таблицю та зображення.
- Оформлено HTML-форму зворотного зв'язку зі станами :hover для кнопок та :focus для полів вводу.
- Зроблено базову адаптацію інтерфейсу під мобільні пристрої за допомогою media queries (для екранів 768px і менше).

## Як запустити
1. Склонуйте репозиторій.
2. Відкрийте файл index.html у будь-якому сучасному браузері.
3. Рекомендовано використовувати розширення Live Server для локального запуску.
